# 14,15,16,17장 js문법(2), 제어문(조건

- boolean

```jsx
console.log(true);
console.log(false);
```

- 비교연산자

```jsx
console.log(1==1);//true
console.log(1==2);//false
console.log(1>2);//false
console.log(1<2);//true
console.log(1===1);//true
console.log(1===2);//false
```

==와 ===다름



- 제어문

시간의 순서에 따라 기계적으로 처리되게 하기 위해 사용

- 조건문

```jsx
if(true){
console.log('c1');
}
else{
console.log('c2');
}
```
